# Citrix Platform Automation

PowerShell tooling for automating and validating enterprise Citrix, Entra, Intune, and Azure platform components.

## Structure
- `scripts/citrix` – Citrix Cloud & platform automation
- `scripts/entra` – Entra ID / identity automation
- `scripts/intune` – Intune device & policy automation
- `scripts/azure` – Azure platform helpers

## Notes
Scripts are designed for enterprise environments and prioritize clarity, auditability, and predictable behavior.
